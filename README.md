# **🎓 합격의 신 : 나는 어디로? (2026 대입 시뮬레이션)**

**"내 성적으로 갈 수 있는 대학은 어디일까?"** \> 2026학년도 대학 입시를 미리 체험해보는 **실전형 대입 시뮬레이션 게임**입니다.

## **📖 프로젝트 소개**

\*\*\[합격의 신\]\*\*은 실제 입시 데이터를 기반으로 구성된 웹 시뮬레이션 게임입니다.

플레이어는 자신의 **내신**과 **모의고사 성적**을 입력하고, 퀴즈와 대학별 고사를 거쳐 **수시 6장**과 **정시 3장**의 원서를 접수하며 최종 합격을 목표로 합니다.

단순한 운이 아닌, \*\*전형별 특징(교과/종합/논술)\*\*과 \*\*지원 전략(상향/적정/안정)\*\*이 중요한 전략 게임입니다.

## **🎮 주요 기능 (Key Features)**

### **1\. 🌏 방대한 대학 데이터베이스**

* **총 50개 주요 대학 탑재:**  
  * **SKY (최상위권):** 서울대, 연세대, 고려대  
  * **인서울 주요대:** 서강대, 성균관대, 한양대, 중앙대, 경희대 등 20여 곳  
  * **지거국 (지방거점국립대):** 부산대, 경북대, 전남대, 충남대, 강원대 등 10곳  
  * **지역 주요 대학:** 울산대, 영남대, 동아대 등

### **2\. ⚡ 성적 향상 퀴즈 (Combo System)**

* 국어, 수학, 영어, 사회, 과학, 시사 등 **다양한 영역의 30+ 문제 은행**  
* **콤보(Combo) 시스템:** 연속 정답 시 내신/수능 등급 대폭 상승\! (틀리면 감점 😭)

### **3\. 📝 현실적인 수시 지원 (6장)**

* **전형별 맞춤 로직:**  
  * **학생부 교과:** 오직 **내신 성적**으로 칼같이 판정 (운 요소 배제)  
  * **학생부 종합:** 서류 \+ **면접 미니게임** 결과 반영  
  * **논술 전형:** 부족한 내신을 \*\*논술 고사(고난도 문제)\*\*로 뒤집기 가능  
* **유연한 원서 접수:** 1장부터 6장까지 원하는 만큼 지원 가능

### **4\. 🎤 대학별 고사 미니게임**

* **면접관과의 대화:** 인성, 지원동기, 시사 이슈 등 랜덤 질문에 답변하여 합격률 UP  
* **고난도 논술:** 인문/수리/상경 등 계열별 심화 문제 풀이

### **5\. 🥁 정시 모집 (가/나/다군)**

* 수시 납치(수시 합격 시 정시 불가) 시스템 구현  
* 수능 백분위 점수를 기준으로 **최종 3장**의 승부수

## **🕹️ 게임 방법 (How to Play)**

1. **프로필 생성:** 이름, 현재 내신(1.0\~9.0), 모의고사 백분위(0\~100) 입력.  
2. **스탯 업:** 5과목의 퀴즈를 풀어 성적을 최대한 끌어올리세요. (연속 정답 필수\!)  
3. **수시 지원:** 50개 대학 중 원하는 곳을 골라 최대 6장까지 원서를 넣으세요.  
   * *Tip: 교과 전형은 안정권, 종합/논술은 상향 지원을 추천합니다.*  
4. **대학별 고사:** 종합/논술 전형 지원자는 면접장과 논술 고사장으로 이동합니다.  
5. **수시 발표:** 합격 시 게임 종료(축하합니다\!🎉), 불합격 시 정시로 이동합니다.  
6. **정시 지원:** 남은 기회는 단 3장\! 수능 성적으로 마지막 합격을 노리세요.
